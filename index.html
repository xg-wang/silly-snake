Abstract

The Snake is a very popular games all over the world. The rule of the game is quite simple as players control the snake body to eat the randomly appearing items. Every time when the snake eats one item, it will increase one length of its own body. In this project, we simplify the learning process and make the game more exciting. We use reinforcement learning algorithm like Q-Learning to train the snake and make it eat more apples in a limited time. That means snake will take actions based on current and previous states. We further use State-Action-Reward-State-Action (SARSA) algorithm to compare with Q-Learning algorithm. The results show that the off-policy Q-Learning algorithm achieves better performance based on our experimental environment.

At the end of project, we combine the usage of reinforcement learning and deep neural networks, called Deep Q-Learning for Q-learning with the function approximation with Neural Networks.
